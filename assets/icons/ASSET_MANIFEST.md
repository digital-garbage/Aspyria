# Inventory icon assets

Vendored for the Aspyria inventory UI.

- `vendor/opengameart-rpg-inventory-icons.png`
  - Source: https://opengameart.org/content/rpg-inventory-icons
  - Author: BizmasterStudios
  - License: CC0
  - Use: original fantasy inventory sheet; the sword is also used for the weapon slot.

- `items/`
  - Use: curated per-slot inventory icons used by the Qt inventory UI.
  - Layout: `items/<slot>/<rarity>.png`.
  - Slots covered: weapon, armor, charm, boots, ring, relic.
  - Rarities covered per slot: common, uncommon, rare, mythical, legendary, corrupted.
  - Older flat files like `weapon.png` are kept only as fallback files.

- `vendor/opengameart-cc0-equipment/`
  - Source: https://opengameart.org/content/cc0-torso-wear-icons
  - Source: https://opengameart.org/content/cc0-footgear-icons
  - Source: https://opengameart.org/content/cc0-jewelry-icons
  - Source: https://opengameart.org/content/cc0-sword-icons
  - Source: https://opengameart.org/content/cc0-mineral-icons
  - Author: AntumDeluge, with bundled source notes from the packs.
  - License: CC0
  - Use: expanded slot and rarity icon coverage.

- `vendor/kenney-game-icons/`
  - Source: https://opengameart.org/content/game-icons
  - Author: Kenney
  - License: CC0
  - Use: UI symbols for inventory/settings affordances.
